1. 📥 INGRESOS
   Crear notas de ingreso con detalles de productos
   • Backend: 9 endpoints
   • Frontend: NotaIngresoList + NotaIngresoForm
   • Características:
     - Crear, listar, actualizar, aprobar
     - Importar desde Excel
     - Exportar a Excel
     - Auto-incremento de números
     - Validación completa

2. 📋 ACTAS DE RECEPCIÓN
   Registrar conformidad de recepción física
   • Backend: 9 endpoints
   • Frontend: ActaRecepcionList + ActaRecepcionForm
   • Características:
     - Crear desde ingreso
     - Comparar cantidades esperado vs recibido
     - Cálculo automático de diferencias
     - Generación de ajustes de stock
     - Aprobación con validación

3. 📦 SALIDAS
   Registrar notas de salida a clientes
   • Backend: 8 endpoints
   • Frontend: NotaSalidaList + NotaSalidaForm
   • Características:
     - Crear con validación de stock
     - Seleccionar cliente y productos
     - Despachar salida
     - Exportar a Excel
     - Descuento automático de stock

4. 📊 KARDEX
   Histórico completo de movimientos
   • Backend: 4 endpoints
   • Frontend: KardexList
   • Características:
     - Ver todos los movimientos
     - Filtro por producto, tipo, fecha
     - Visualización de saldos progresivos
     - Exportar a Excel
     - Búsqueda avanzada

5. ⚠️ ALERTAS
   Sistema inteligente de alertas de vencimiento
   • Backend: 7 endpoints
   • Frontend: AlertasList
   • Características:
     - Resumen visual (vigentes, próximos, vencidos)
     - Filtro por estado
     - Marcar como leída
     - Auto-cálculo de días faltantes
     - Gestión completa de alertas

6. 👥 USUARIOS
   Gestión de usuarios y roles
   • Backend: 10+ endpoints
   • Frontend: UsuariosList
   • Características:
     - CRUD de usuarios
     - Gestión de roles
     - JWT authentication
     - Auditoría de accesos
     - Password cifrado

7. 📈 REPORTES
   Análisis y reportes de datos
   • Backend: 5 endpoints
   • Frontend: ReportesCompleto
   • Características:
     - Stock actual con lotes
     - Ingresos por período
     - Salidas por período
     - Productos por categoría
     - Exportación multi-sheet Excel

OPCIÓN 2: Manual
  1. cd /home/bugsara/CEREX/Proyecto_ALMACEN
  
  2. Preparar BD:
     mysql -u root -p almacen_db < apps/api/migrations/002_create_complete_schema.sql
  
  3. Iniciar API:
     cd apps/api
     npm install
     npm run dev
  
  4. En otra terminal, iniciar Frontend:
     cd apps/electron
     npm install
     npm run dev
  
  5. Abrir Electron y login:
     Usuario: admin
     Contraseña: Admin123!
